Knots & Primes

Linking, triple linking, and zeta functions — on both sides of the mirror. Dotted terms carry definitions — hover, Tab, or click to pin; Enter/Space pins too, Esc closes.

What a green suite does and does not mean. The computational checks recompute the load-bearing numbers from scratch — symbols, roots, Magnus coefficients, matrix orders, residue degrees, Bernoulli residues — and would fail if that arithmetic broke. They do not cover every digit on screen: a few displayed quantities (individual digits of the ζ truncations, some term counts) are asserted only to a tolerance, or not at all. The rendered-text pins assert that the prose on the page still matches what the code produces — they catch drift between the two, and nothing else: if code and prose are wrong together, or a theorem is misattributed, or a citation carries the wrong year, this suite stays green. It tests consistency, not truth.
    Presenting this app
    Keyboard. t anywhere outside a text field runs the self-tests and scrolls them into view — the closing beat, one keystroke. The tab bar is a single tab stop: Tab to it, then ←/→ to move between tabs. Every tooltip is reachable by Tab and opens with Enter.
    Deep links. The URL fragment restores a whole beat, so a talk can be driven from bookmarks rather than clicks: #tab=zeta&open=zxK1361,zxHopf&goto=zetaK1361 — tab selects a tab, open is a comma-separated list of card ids to expand, goto is an element to scroll to. Ids are the ones in the page source; unknown ids are ignored rather than failing.
    Offline. The file is self-contained and works with no network. The only external fetch is the optional 3D view of the Borromean rings, which falls back to a built-in canvas renderer when the CDN is unreachable.
